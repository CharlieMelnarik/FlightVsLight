# FlightVsLight

Standalone Next.js website for visualizing a flight route against daylight, darkness, and time zones.

## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Recommended deployment
Use Vercel for the smoothest Next.js deployment, then attach `flightvslight.com` as the custom domain.
